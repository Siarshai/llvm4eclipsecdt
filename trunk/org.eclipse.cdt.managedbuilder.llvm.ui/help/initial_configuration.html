Initial configuration

When you launch an Eclipse workspace with LLVM plugin for the first time, you need to set the paths to LLVM binaries, headers and libraries. This is done in Eclipse by navigating to Window -> Preferences -> LLVM. Set LLVM installation folder to point where your LLVM binaries are, include directory to point where your Clang includes are and library path to the directory where your LLVM libraries reside. After the paths are set, you are ready to use the LLVM plugin.

If your paths in Preferences and LLVM are correctly set and the plugin is able to find your LLVM binaries, you should have the option to use "LLVM with Clang (Linux/Windows)" toolchain and "LLVM with GCC (Linux/Windows) toolchain". The toolchains are available in project types of 'executable', 'static library' and 'dynamic library' in both C and C++.

An alternative way is to set the LLVM binary path to PATH system environment variable.

You may also have to include the path to libstdc++ (Standard C++ Library) in order to compile C++ programs by adding the path in LLVM Preferences or appending it to LD_LIBRARY_PATH system environment variable.
